# Stock Market App (Flet + Python)

A simple desktop application built with **Python** and **Flet** that allows users to fetch and visualize stock market data using the **Alpha Vantage API**.

The app displays:
- Latest stock prices (Open, High, Low, Close)
- Historical closing prices as a line chart
- Selectable time ranges (1 week to 5 years)

---

## Features

- Desktop UI
- Real-time stock data
- Interactive line chart for visualization
- Multiple time range selection (from 1 week to 5 years)
- Error handling for invalid symbols or API issues
